**31 (UPT Coin Game)**

- 2-6 players, 3 coins each. Pot to last survivor.
- 32-card deck (7-A, 4 suits).
- Dealer deals 3 cards to each + 1 blind set of 3.
- Dealer peeks only one set, chooses to keep or take unseen. Rejected 3 = face-up table.
- Play starts left of dealer.
- Turn: Swap 1 (hand↔table), Swap all 3, Pass, or Knock.
- After any swap: Fire (3 Aces=33) or 31 → instant end.
- 3-of-a-kind (non-Aces=30.5) strong but no instant end.
- Knock: one full round after, then end.
- End scoring:
  - Fire: all others pay 1 to pot.
  - Else: all lowest score(s) pay 1 each (ties pay).
- Scores: Fire=33, 31, 3-kind(non-A)=30.5, else max one suit (A=11, 10/J/Q/K=10, 7-9=face).
- Rotate dealer left. Out at 0 coins.
